О музыке

Музыка написана мной, но основу я искал в произведении Ференца Листа «Грёзы любви» (Ноктюрн №3). Взяв за основу фрагмент его нот, я создал код для Strudel REPL, который потом записал в MP3.

Что такое Strudel REPL?

Strudel REPL — это онлайн-площадка для создания музыки с помощью кода. Представ себе это как музыкальный конструктор, где вы собираете звуки, записывая простые команды на языке JavaScript.

Как это работает?

  • Вы пишете код (например, ноту или ритм)
  • Нажимаете кнопку воспроизведения
  • Сразу же слышите, что у вас получилось

Самое главное — вы можете менять код на лету, и музыка будет меняться в реальном времени, без остановки. Это похоже на живую импровизацию, но вместо инструмента у вас — клавиатура.

Посетить Strudel REPL

Код, который я написал для этой музыки:

// Настройка тональности (оригинал - Ля-бемоль мажор)
scale('ab major')

// Основная мелодия
const melody = note(`
[eb5] [eb5] [eb5] [eb5] [eb5] [eb5]
[f5] [eb5] [db5] [c5] [db5] [eb5]
[ab4] [ab4] [ab4] [ab4] [ab4] [ab4]
[bb4] [ab4] [g4] [f4] [g4] [ab4]
[eb5] [eb5] [eb5] [eb5] [eb5] [eb5]
[f5] [eb5] [db5] [c5] [db5] [eb5]
[ab5] [g5] [f5] [eb5] [db5] [c5]
[bb4] [ab4] [g4] [f4] [eb4] [db4]
`).slow(10).gain(0.4)

// Аккомпанемент
const accompaniment = note(`
[ab3 eb4 ab4] [ab3 eb4 ab4] [ab3 eb4 ab4] [ab3 eb4 ab4]
[bb3 f4 bb4] [bb3 f4 bb4] [ab3 eb4 ab4] [ab3 eb4 ab4]
[eb3 bb3 eb4] [eb3 bb3 eb4] [ab3 eb4 ab4] [ab3 eb4 ab4]
[db3 ab3 db4] [db3 ab3 db4] [c3 g3 c4] [c3 g3 c4]
[ab3 eb4 ab4] [ab3 eb4 ab4] [ab3 eb4 ab4] [ab3 eb4 ab4]
[bb3 f4 bb4] [bb3 f4 bb4] [ab3 eb4 ab4] [ab3 eb4 ab4]
[eb3 bb3 eb4] [eb3 bb3 eb4] [ab3 eb4 ab4] [ab3 eb4 ab4]
[db3 ab3 db4] [db3 ab3 db4] [c3 g3 c4] [c3 g3 c4]
`).slow(60).gain(0.3)

// Собираем композицию с реверберацией
stack(
melody.room(0.8).size(0.7),
accompaniment.room(0.6)
)
